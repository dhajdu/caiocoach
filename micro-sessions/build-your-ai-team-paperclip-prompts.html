Prompt Library

Copy-ready prompts to help you complete and extend this session's work.

PROMPT 1
Research Agent Brief
Use this to give a research agent a specific intelligence task.
Prompt 
You are a research agent working on [project name / launch / initiative].

Your job is to find and summarize the following:
- [Research question 1 - e.g., competitor pricing in [market]]
- [Research question 2 - e.g., key customer segments and their main concerns]
- [Research question 3 - e.g., the top three risks in entering this market]

For each item, write a short paragraph with the most important finding and one concrete data point if available.

End with a one-paragraph summary of the single biggest insight across all three areas.
PROMPT 2
Copy Agent Brief
Use this to give a copy agent a content creation task connected to a campaign or launch.
Prompt 
You are a copy agent working on [campaign name / product name].

Your audience is [describe audience - e.g., retail partners in Southeast Asia / internal leadership / new customers].

Write the following:
- [Deliverable 1 - e.g., a launch email to retail partners, under 200 words]
- [Deliverable 2 - e.g., three social media posts for the first week of launch]
- [Deliverable 3 - e.g., a one-paragraph pitch for an internal presentation]

Tone: [e.g., direct and energetic / professional and concise / conversational and warm]

Do not use jargon. Write for someone who is reading quickly and deciding whether to act.
PROMPT 3
Analytics Agent Brief
Use this to give an analytics agent a data interpretation task.
Prompt 
You are an analytics agent. The files in this folder contain [describe data - e.g., pilot sales data / customer survey results / weekly performance reports].

Do the following:
- Identify the top [3-5] patterns or trends in the data
- Flag any result that is significantly above or below what you would expect
- Write a one-paragraph executive summary I can use in a [leadership meeting / board presentation / team briefing]

Keep the summary plain and direct. No jargon. Write it for a decision-maker who has 60 seconds to read it.
PROMPT 4
Goal Statement for Paperclip
Use this to write the top-level goal that coordinates all your agents.
Prompt 
Goal: [One clear sentence on what you are trying to achieve and by when]

Context: [Two to three sentences on the situation - what is happening, what is at stake, and what success looks like]

Constraints:
- Budget per agent: $[X]
- Timeline: [date or timeframe]
- Output format: [e.g., written documents / slide-ready summaries / structured data]

Agents on this team: [list agent names and roles]
PROMPT 5
Agent Review Prompt
Use this after you receive an output from a Paperclip agent to assess quality and decide whether to run again.
Prompt 
I received the following output from my [research / copy / analytics] agent:

[Paste agent output here]

My goal was: [paste your goal]

Tell me:
1. What is strong about this output?
2. What is missing or weak?
3. What one change to the agent brief would most improve the next run?
