03 Reframe

What Claude Code Actually Is

The name says "code." So everyone assumes it is for developers. It is not.

Claude Code is a version of Claude that runs in your terminal and works directly with your files and computer. It does not require you to write code. It requires you to describe tasks in plain English - and it writes and runs the code itself.

What people think it is

A tool for software engineers to write and debug code. Not relevant to non-technical roles.

What it actually is

An AI that can operate your file system, batch-process documents, rename files, generate reports, and automate repetitive tasks - all from plain-language instructions.

Key Insight

Claude Code is not a coding tool. It is an instruction tool that happens to live in a terminal. The distinction matters because it changes who can use it.

04 Capability

What It Can Do for You Right Now

Here are the things Claude Code can do for a non-technical AI Officer today - no code required from you.

1

Batch rename and organize files

Rename hundreds of files according to a naming convention, sort them into folders by date or category, or clean up a messy download folder in seconds.

2

Process and summarize documents

Read every file in a folder, extract key information, and produce a consolidated report - without opening a single file yourself.

3

Convert file formats

Convert CSV to structured markdown, text files to formatted reports, or raw data dumps into readable summaries.

4

Generate and run reports

Pull data from files in a folder, compute totals, find patterns, and output a formatted summary - on demand or on a schedule.

The pattern

You describe the task in plain English. Claude Code writes the code, runs it, and shows you the result. You review. You do not need to understand the code. You need to understand the task.

05 Exercise

Open Your Terminal

Everyone does this at the same time. Two minutes.

Do this right now

Open your terminal. On Mac: Command + Space, type "Terminal," enter. On Windows: Windows key, type "cmd," enter. Type "ls" on Mac or "dir" on Windows and press enter. You will see a list of files. That is your computer talking back to you. Post what you see in the chat.

If you already have Claude Code installed, type "claude" and press enter. If not, you will install it in the challenge this week.

What to notice

The terminal is not scary. It is a list of files and a blinking cursor. Everything else is just knowing what to type next - and Claude Code handles most of that for you.

06 Decision Guide

Cowork vs Claude Code: When to Use Each

Both Cowork and Claude Code work with files. They are not the same tool.

Use Cowork when

You are working with structured document tasks - briefings, summaries, reports, organized outputs. Cowork has a UI. Your team can run it without the terminal. Good for recurring tasks with a consistent format.

Use Claude Code when

You need to operate directly on files - rename, convert, batch-process, generate from raw data. Claude Code is faster for ad hoc tasks and can handle files at scale. Good for one-time jobs and technical file operations.

Key Insight

Cowork is your document workflow tool. Claude Code is your file operation tool. Both are useful. The choice depends on whether your team needs a UI or whether speed and scale matter more.

07 Leadership

The Leadership Layer

Here is what changes when your team can use Claude Code. Every manual file operation that currently takes 30 minutes becomes a 30-second command. More importantly, those commands can be documented and handed to anyone on the team.

The AI Officer move is not to use Claude Code yourself. It is to build a library of Claude Code commands that your team can run for their most common tasks - with no technical knowledge required from them.

Key Insight

Claude Code is not a developer tool. It is a door. Once you walk through it, every powerful AI automation tool is on the other side - and you can bring your whole team with you.
