Agentic AI

Your First Cowork Workflow

Agentic AI Essentials · Hands-on micro-session

Session Objective

95 percent of organizations investing in AI get no measurable return. Not because the tools do not work - the tools are extraordinary right now. Because they skip the process and go straight to the tool.

Your Challenge
Complete the following steps
STEP 1
Define - Map your workflow with Claude
Choose one recurring task that involves files. It should be something you do every week or month that you find repetitive - not because it is hard, but because it is the same every time.
STEP 2
Discover - Connect your folder and find out what Claude needs
Open Cowork. Click Select Folder and connect a real folder from your work - wherever the files for this task actually live.
STEP 3
Design - Ask Claude to write the instructions
Do not write the project instructions yourself. Tell Claude what you want the workflow to do and ask it to write the instructions. Use Prompt 3 from the prompt library.
STEP 4
Determine - Set your ROI baseline, then run it
Before you click send, calculate your baseline. How long does this task take you today? How often? Use Prompt 4 from the prompt library to get a real number.
STEP 5
Deploy - Confirm the system runs without you
Close the session and reopen it. Run the workflow again from scratch - without explaining the context, without pasting anything new. Does it work? If yes, the system is deployed. The instructions carry the knowledge.…
